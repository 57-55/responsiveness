Responsive web design

Small

What is responsive web design? (content for smartphones)

Responsive web design makes your web page look good on all devices. It uses only HTML and CSS. It is not a program or a JavaScript.

Medium

Who invented responsive web design? (content for tablets)

Coined by Ethan Marcotte in May 2010, responsive web design (RWD) is idea that “[website] design and development should respond to the user's behavior and environment based on screen size, platform and orientation”.

Large

What are the advantages and benefits of responsive web page design? (content for pc)

Access to content that is appropriately adapted on any device greatly improves the user experience. A good responsive design also improves readability, increases the time spent on a website, it enhances interaction or, in the case of e-commerce, improves sales.
